1. Pull new code
2. Rebuild Docker image
3. Stop old container
4. Remove old container
5. Create new container
6. Start new container



| Step                    | Command/Action                                                  |
| ----------------------- | --------------------------------------------------------------- |
| 1. Pull new code        | `git pull origin main` → updates code on server                 |
| 2. Rebuild Docker image | `docker compose up --build` → rebuilds images from Dockerfile   |
| 3. Stop old container   | Docker stops running containers that need to be updated         |
| 4. Remove old container | Docker removes the old containers before recreating them        |
| 5. Create new container | Docker creates new containers from the rebuilt images           |
| 6. Start new container  | Docker starts the new containers (`-d` runs them in background) |
